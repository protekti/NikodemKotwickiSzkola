Wyjaśnienie kodu źródłowego w Pythonie

Linia 1

print("*** Program powitalny ***")

Ta linia wyświetla na ekranie tekst „*** Program powitalny ***”. Funkcja print() służy do wypisywania tekstu.

Linia 2

login = input("Podaj nazwę użytkownika: ")

Program wyświetla komunikat „Podaj nazwę użytkownika:”, a następnie czeka, aż użytkownik wpisze tekst. Wprowadzona wartość jest przypisywana do zmiennej login.

Linia 3

print()

Wyświetla pustą linię, aby oddzielić teksty na ekranie.

Linia 4

print("Użytkownik " + login + " został programistą języka Python")

Program łączy tekst „Użytkownik” oraz zawartość zmiennej login z tekstem „został programistą języka Python” i wyświetla całość.

Linia 5

print()

Ponownie wyświetla pustą linię, aby zakończyć program w bardziej przejrzysty sposób.

Cały kod do skopiowania

print("*** Program powitalny ***")
login = input("Podaj nazwę użytkownika: ")
print()
print("Użytkownik " + login + " został programistą języka Python")
print()
